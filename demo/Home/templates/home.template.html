angular-webrtc
Description

angular-webrtc is a simple wrapper for the WebRTC APIs, enabling you to use it on Google Chrome or Mozilla Firefox without having to care about prefixes.

Install

Install it using Bower :

bower install ac.angular-webrtc

Include angular-webrtc in your html page :

<script src="bower_components/ac.angular-webrtc/dist/angular-webrtc.min.js" type="text/javascript"></script>

Inject ac.angular-webrtc in your app :

angular.module('your.app', ['ac.angular-webrtc']);
